# 小导航

### demo 运行依赖

```
npm install
```

### 以下终端命令行均可通过

    -> 打开 vscode 侧边栏
    -> 点击资源管理器(第一个图标)
    -> 左下角的 NPM 脚本
    -> 执行相关命令即可

#### 本地运行预览命令行

```
npm run serve
```

#### 本地打包命令行

```
npm run build
```

### 本地端口部署命令行

http-server -p 8888 ./dist
